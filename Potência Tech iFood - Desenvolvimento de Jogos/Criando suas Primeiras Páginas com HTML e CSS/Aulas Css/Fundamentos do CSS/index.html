No CSS nós possuimos seletores por ID

  • Seletor por Tipo: Busca elementos por uma tag HTML específica
  • Seletor por ID(#): Busca elementos através do atributo "id"
  • Seletor por Classe: Busca elementos através do atributo "class"
  • Seletor por universal: Seleciona todos os elementos HTML
  • Seletor de atributo: Seleciona elementos quei possuem um atributo especifico em nosso documento HTML e conseguimos também buscar atributos com um valores especifico
